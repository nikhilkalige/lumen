Google API Key Tester

Use this tool to verify if your Google API Key is valid and configured correctly. This test uses the Google Books API. 


        


        

             
Instructions

             
    
                
  1. Go to your Google Cloud Console API Library.
    2. 
                
  2. Search for and **enable the "Google Books API"** for your project. Your key won't work without this step.
    3. 
                
  3. Go to the Credentials page and copy your API Key.
    4. 
                
  4. Paste the key into the input box above and click the "Test API Key" button.
    5.
